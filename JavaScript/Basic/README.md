## 상수 const
- 담긴 값이 바뀔 수 없다. 
- 흔히 "대문자"로 명명한다. 
- 상수를 선언과 동시에 초기화해야한다 << 선언하기만 하는건 불가하다.

- **값이 바뀔 일이 없는 데이터는 상수로 선언해라**
